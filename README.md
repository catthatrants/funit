# FUnit - A Blooket-like Quiz Game

FUnit is an interactive quiz game built with HTML, CSS, and JavaScript. It's designed to make learning fun and engaging!

## Features

- 🎮 **Multiple Game Modes**
  - Classic: Answer questions to earn points
  - Timed: Race against the clock
  - Multiplayer: Compete with friends (coming soon)

- 📝 **Quiz Creation**
  - Create custom quizzes
  - Add multiple choice questions
  - Save quizzes locally

- 🎨 **Beautiful UI**
  - Modern gradient design
  - Smooth animations
  - Responsive on all devices

- 📊 **Game Statistics**
  - Track your score
  - View accuracy percentage
  - See performance metrics

## Getting Started

1. Clone the repository
   ```bash
   git clone https://github.com/catthatrants/funit.git
   cd funit
   ```

2. Open `index.html` in your browser
   ```bash
   # On macOS
   open index.html
   
   # On Linux
   xdg-open index.html
   
   # Or simply drag and drop the file into your browser
   ```

## How to Play

1. **Start a Game**: Click "Start Game" on the home screen
2. **Select Mode**: Choose between Classic, Timed, or Multiplayer
3. **Answer Questions**: Click on the correct answer
4. **Earn Points**: Get 10 points for correct answers, lose 5 for incorrect
5. **View Results**: See your final score and accuracy

## Creating Your Own Quiz

1. Click "Create Quiz" from the home screen
2. Enter quiz title and description
3. Add questions and answer options
4. Mark the correct answer
5. Click "Save Quiz" to store it locally
